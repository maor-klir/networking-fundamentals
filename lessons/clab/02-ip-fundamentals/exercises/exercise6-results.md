## Explain the difference between local and remote subnets, and why gateways are needed

- A local subnet is when the destination IP address falls within the same network as the host's own IP and subnet mask.
- A remote subnet is whena a destination IP address falls outside the host's own network prefix.
- A gateway is needed to reach the remote subnet and a default route will tell the host where to send traffic for any network that falls outside the host's own network prefix.
